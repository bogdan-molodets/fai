#### Install & Update
-------------

Run  _faiSetup_ file or open terminal in project directory and write command

##### Ubuntu
***
```
bash faiSetup.sh
```
##### MacOS
***
```
bash faiSetup 
```
#### Run
-------------
##### Ubuntu
***
`````
bash runFai.sh
`````
##### MacOS
***
`````
bash runFai
`````
